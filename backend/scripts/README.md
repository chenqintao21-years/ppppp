# 高德地图数据导入脚本使用说明

## 功能说明

这个脚本可以从高德地图API自动导入热门城市和景点数据到数据库中。

## 使用步骤

### 1. 确保数据库已启动并配置正确

检查 `.env` 文件中的数据库配置：

```env
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的密码
DB_NAME=tripadvisor
```

### 2. 确保高德地图API密钥已配置

在 `backend/services/mapService.js` 中已经配置了默认的高德地图API密钥。如果需要使用自己的密钥，可以在 `.env` 文件中添加：

```env
AMAP_KEY=你的高德地图API密钥
```

### 3. 运行导入脚本

在 `backend` 目录下执行：

```bash
node scripts/importFromAmap.js
```

### 4. 查看导入结果

脚本会自动：
1. 导入12个热门城市（北京、上海、广州、深圳、杭州、成都、西安、重庆、南京、武汉、厦门、苏州）
2. 为每个城市搜索并导入约20个热门景点
3. 自动生成评分、评论数等数据

## 导入的数据包括

### 目的地（destinations表）
- 城市名称（中文和英文）
- 国家和地区
- 封面图片
- 评分和浏览量

### 景点（attractions表）
- 景点名称
- 描述
- 评分和评论数
- 价格（随机生成）
- 位置信息（城市、国家、经纬度）
- 游览时长
- 取消政策
- 图片

## 注意事项

1. **API限制**：高德地图API有调用频率限制，脚本中已添加延迟避免请求过快
2. **数据更新**：如果数据已存在，脚本会更新而不是重复插入
3. **图片**：目前使用Unsplash的占位图片，实际项目中应该使用真实图片
4. **价格**：价格是随机生成的，实际项目中应该从真实数据源获取

## 自定义城市列表

如果需要导入其他城市，可以修改 `importFromAmap.js` 中的 `POPULAR_CITIES` 数组：

```javascript
const POPULAR_CITIES = [
    { name: '城市名', name_en: 'City Name', country: '中国', region: '地区' },
    // 添加更多城市...
];
```

## 故障排除

### 问题1：数据库连接失败
- 检查MySQL服务是否启动
- 检查 `.env` 文件中的数据库配置是否正确

### 问题2：高德地图API调用失败
- 检查网络连接
- 检查API密钥是否有效
- 查看控制台错误信息

### 问题3：导入的景点数量少
- 高德地图API返回的数据量可能因城市而异
- 可以调整搜索关键词或增加搜索页数

## 后续步骤

导入数据后，可以：

1. 启动后端服务：`npm run dev`
2. 启动前端服务：在 `frontend` 目录运行 `npm run dev`
3. 访问 `http://localhost:5173` 查看首页
4. 首页会自动显示导入的热门目的地和景点
5. 点击景点卡片可以查看详情页面
