## 修复的问题

### 1. 景点详情 API
**问题**: 查询评论时使用了错误的字段名 `attraction_id`
**修复**: 更新为使用 `entity_type` 和 `entity_id`
```javascript
// 修复前
WHERE r.attraction_id = ?

// 修复后
WHERE r.entity_type = ? AND r.entity_id = ?
```

### 2. 收藏功能 API
**问题**: 插入/删除收藏时使用了错误的字段名 `attraction_id`
**修复**: 更新为使用 `entity_type` 和 `entity_id`
```javascript
// 修复前
INSERT INTO favorites (user_id, attraction_id) VALUES (?, ?)

// 修复后
INSERT INTO favorites (user_id, entity_type, entity_id) VALUES (?, ?, ?)
```

### 3. 测试数据初始化
**问题**: 外键约束导致评论插入失败
**修复**: 先查询实际的用户 ID，然后使用正确的 ID 插入评论
