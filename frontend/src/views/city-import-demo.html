🌍 城市自动导入功能 - 使用示例

这个功能允许你搜索任意城市，并自动从高德地图获取该城市的酒店、景点、餐厅信息， 然后导入到数据库中。支持手动搜索城市和自动定位当前位置。

示例 1: 打开城市搜索模态框

点击按钮打开城市搜索对话框，输入城市名称进行搜索

import { citySearchModal } from '../components/citySearchModal.js'; // 打开模态框 citySearchModal.open((data) => { console.log('导入成功:', data); // data 包含: city, location, imported (hotels, attractions, restaurants) });

示例 2: 获取当前位置

使用高德地图API获取用户当前位置

定位结果:
import { amapLocationService } from '../utils/amapLocation.js'; // 获取当前位置 const result = await amapLocationService.getCurrentPosition(); console.log('当前位置:', result); // result 包含: city, province, position {lng, lat}, address

示例 3: 直接导入城市数据

不打开模态框，直接调用API导入指定城市的数据

导入结果:
import { cityAutoImportService } from '../services/cityAutoImport.js'; // 导入指定城市 const response = await cityAutoImportService.autoImportCity({ city: '北京' }); if (response.success) { console.log('导入成功:', response.data); // response.data.imported: {hotels: 50, attractions: 50, restaurants: 50} }

示例 4: 搜索城市建议

输入关键词获取城市搜索建议

搜索结果:
import { cityAutoImportService } from '../services/cityAutoImport.js'; // 搜索城市 const response = await cityAutoImportService.searchCity('北京'); console.log('搜索结果:', response.data);

示例 5: 集成到现有页面

在酒店、景点、餐厅页面添加"导入城市数据"按钮

<button id="importCityBtn">📥 导入城市数据</button> <script type="module"> import { citySearchModal } from '../components/citySearchModal.js'; document.getElementById('importCityBtn').addEventListener('click', () => { citySearchModal.open((data) => { // 导入成功后刷新页面数据 location.reload(); }); }); </script>
