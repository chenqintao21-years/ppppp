🔍 统一搜索组件

支持实时搜索建议、搜索历史、热门搜索、多类型搜索的完整解决方案

基础示例

最简单的使用方式，支持全类型搜索

// HTML <div class="search-bar-component" id="search-basic"></div> // JavaScript const searchBar = new SearchBar({ container: '#search-basic', placeholder: '搜索餐厅、景点、酒店' });

餐厅专用搜索

限定搜索类型为餐厅，不显示类型筛选器

const restaurantSearch = new SearchBar({ container: '#search-restaurant', type: 'restaurants', placeholder: '搜索餐厅', showTypeFilter: false });

景点专用搜索

限定搜索类型为景点

const attractionSearch = new SearchBar({ container: '#search-attraction', type: 'attractions', placeholder: '搜索景点玩乐', showTypeFilter: false });

自定义回调函数

监听搜索和选择事件，自定义处理逻辑

const customSearch = new SearchBar({ container: '#search-custom', onSearch: function(query, type) { console.log('搜索:', query, type); // 自定义搜索逻辑 }, onSelect: function(item) { console.log('选择:', item); // 自定义选择逻辑 } });

功能特性

⚡ 实时搜索建议

输入时自动显示相关建议，支持防抖优化

📝 搜索历史

自动保存搜索历史，支持删除和清空

🔥 热门搜索

显示最近7天的热门搜索词

🎯 多类型搜索

支持餐厅、景点、酒店的统一搜索

⌨️ 键盘导航

支持上下键选择，Enter确认，Esc关闭

📱 移动端适配

响应式设计，支持移动端全屏模式

🎨 高亮显示

搜索结果中关键词高亮显示

🌙 深色模式

自动适配系统深色模式

配置选项

new SearchBar({ // 必需参数 container: '.search-bar-component', // 容器选择器 // 可选参数 placeholder: '搜索...', // 占位符文本 type: 'all', // 搜索类型: all, restaurants, attractions, hotels apiBaseUrl: 'http://localhost:3000/api', // API 基础 URL debounceDelay: 300, // 防抖延迟（毫秒） minSearchLength: 2, // 最小搜索长度 maxHistoryItems: 10, // 最大历史记录数 showHistory: true, // 显示搜索历史 showTrending: true, // 显示热门搜索 showTypeFilter: true, // 显示类型筛选器 // 回调函数 onSearch: function(query, type) {}, // 搜索回调 onSelect: function(item) {} // 选择回调 });
